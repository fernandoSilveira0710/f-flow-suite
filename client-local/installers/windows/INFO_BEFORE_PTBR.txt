F-Flow Suite — Client Local (Instalador em Português)

Descrição do projeto
O Client Local da F-Flow Suite instala dois serviços Windows para uso local:
- F-Flow Client Local Service: API local e serviços complementares (porta 8081)
- F-Flow ERP Local Service: servidor estático do ERP local (porta preferencial 8080, com fallback automático para 8088, 8090 ou 8082)

O que será instalado
- Executável principal: f-flow-client.exe
- Serviço Windows via WinSW para o Client Local
- Serviço Windows via WinSW para o ERP Local (erp-server.exe)
- Frontend ERP (arquivos em "erp\\dist")
- Arquivo .env com configurações de portas e caminhos
- Prisma Client e biblioteca de engine (query_engine-windows.dll.node)
- Diretórios de logs em C:\ProgramData\FFlow\logs\client-local e \erp
- Atalhos no Menu Iniciar e área de trabalho

Portas utilizadas
- ERP: porta preferencial 8080. Se 8080 estiver indisponível, o sistema escolhe automaticamente 8088, 8090 ou 8082. Use o atalho "Abrir ERP" (ele detecta e abre a primeira porta disponível).
- Client Local (licenciamento e saúde): http://localhost:8081/licensing/status

Requisitos
- Windows 64 bits
- Privilégios de administrador para instalar serviços

Observação
Durante a instalação, os serviços são instalados e iniciados automaticamente. Os logs ficam em C:\ProgramData\FFlow\logs.
